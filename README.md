# Quiz.py - Quiz de Python para TestlT

¡Bienvenido al quiz interactivo de Python creado para evaluar habilidades técnicas (hard skills) en procesos de selección!

## Descripción

Este es un quiz de consola desarrollado en Python puro que contiene preguntas de opción múltiple sobre conceptos fundamentales y avanzados del lenguaje. Ideal para entrevistas técnicas, prácticas de estudio o evaluación rápida de conocimientos.

## Características

- Totalmente interactivo en la terminal
- Sin dependencias externas (solo Python estándar)
- Preguntas sobre sintaxis, estructuras de datos, OOP, funciones, excepciones y más
- Cálculo automático de puntaje final
- Feedback inmediato por pregunta

## Requisitos

- Python 3.6 o superior

## Instalación

```bash
git clone https://github.com/brayanvillca/Quiz.py.git
cd Quiz.py
```

# Cómo ejecutar
```bash
python quiz.py
```
¡Listo! El quiz comenzará automáticamente.
